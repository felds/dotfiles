# Dotfiles

Collection of configuration files for shell, editor, and tools.


## Usage

Clone this repository, preferably under `$HOME` and use the [stow](https://www.gnu.org/software/stow/) tool to create links to these dotfiles:

```sh
git clone --depth=1 git@github.com:felds/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
stow
```
